NSE Options Data Pipeline

Clean, analysis-ready NIFTY options data for quantitative research and algo trading.

What This Repository Provides

  • 5+ years of cleaned NIFTY options data (sample included)
  • Implied Volatility (IV) and Greeks (delta, gamma, theta, vega, rho) for every contract
  • Execution cost modeling (bid-ask spread + slippage) for realistic backtesting
  • Tradability filters (liquidity, open interest, volume) to isolate executable trades
  • Fast query API (FastAPI starter) for strategy development
  • Example strategies (Iron Condor) and notebooks for quick start

Why Use This Dataset?

Raw NSE data lacks IV, Greeks, and realistic execution costs. Building this pipeline yourself requires:

  • Multi-source data ingestion (NSE, Yahoo Finance, interest rates)
  • Complex numerical methods for IV and Greeks calculation
  • Execution modeling calibrated to market impact
  • Data cleaning and validation at scale

This repository gives you a reference implementation, while production-grade data and API access are available on request.

Quick Start

# Clone the repository
git clone https://github.com/darshkale/nse-options-data-pipeline.git
cd nse-options-data-pipeline

# Install dependencies
pip install -r requirements.txt

# Run the strategy demo (see results in under 1 minute)
python examples/run_strategy.py

# Explore the demo notebook
jupyter notebook notebooks/demo_iron_condor.ipynb
